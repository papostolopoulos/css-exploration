Typograpy

In this section we have modified the fonts of the page so we can distinguish different elements, make the page easier and more pleasing to read.

The attributes that we can modify in font settings are:

  • The color:  h1 {color: black;} 
  • The font size:  h2 {font-size: 22px;} 
  • The font style:  h3 {font-style: italic;} 
  • The font variant:  .logo {font-variant: small-caps;} 
  • The font weight:  h4 {font-weight: lighter;} 
  • The line height:  body {line-height: 18px;} 

On some cases, the web developers use a combination of the above options and include them all (except the color) in one line. This shorthand option has the order of  font-style ,  font-variant ,  font-weight ,  font-size ,  line-height , and  font-family .
Example:  body {font: normal normal 400 14px/22px "Open Sans", Helvetica, Arial, sans-serif} 

How was CSS used

  • We set up different fonts for different elements of the page
  • We modified the size of the different text properties
  • We modified the colors of the different fonts
  • We set up the height between different lines of text
  • We set up the fonts weight in case we needed to emphasize specific text.
