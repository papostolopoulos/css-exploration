HTML/CSS history

(by David Shin)

In the early days of the world wide web (WWW), documents or other information were often sent as simple plain text across the wires that formed the internet. Because the original intent was to allow scientists to share information at the particle accelerator CERN, Sir Tim Berners-Lee, the inventor of the WWW, developed the Hyper Text Markup Language (HTML). HTML included the means to allow some basic formatting of the documents being shared.

Once styling became more complex and demanding, there was a need for a way that would allow the separation of the code that related to the styling of the page from the rest of the document. CSS was introduced in order to satisfy this need. CSS files are used to control the attributes of a webpage, while the overall style of an html page is being managed through separate files.

By including some type of order and precedence we can keep CSS code DRY (don't repeat yourself). This increases readability for the developer while reducing the memory requirements on both the client and the server side, as well as during the transmission process.

CSS Syntax

CSS has a syntax that has some similarities to the way JSON files or JavaScript is structured. The main components of CSS are the following:

  • A selector: is the one that points to the HTML element (or elements) that you want to style. If this would be stated in plain English, it would read like: I want you to select the h1 tags.
  • Declarations: These are the different properties that state the type of styling you want a particular element to have and what the value is. If this would be stated in plain English, it would read something like: I want to select the h1 tags. (selector) and I want their color to be green.
  • Property: The property is the entity that states what needs to change (eg: color, positioning, background, display)
  • Value: The value is the entity that states how the property changes / what the value of the property is. If this would be stated in plain English, it would read something like: I want to select the h1 tags. (selector) and I want the following things to happen:
    • Declaration: I want the - property - color to have a value green
    • Declaration: I want the - property - font-size to have a value of 16px.
