What is CSS?

CSS stands for Cascading Style Sheets. CSS is a style language sheet that is being used to modify the way an HTML page is being presented. Let's take a random person as an example. This person might wear a coat, a skirt, and a hat one day, and jeans, a t-shirt and glasses the other day. The person is always the same but the way she looks is different based on the clothes she is wearing. Likewise, CSS can give a different look, feel and level of interactivity to the different components of the HTML page, based on the different properties we are modifying to the elements (tags) of the HTML page.

Syntax structure

CSS has a syntax that has some similarities to the way JSON files or JavaScript object literals are structured. The main components of CSS are the following:

  • A selector: The selector is the one that points to the HTML element (or elements) that you want to style. If this would be stated in plain English, it would read like: I want you to select the h1 tags.
  • Declarations: These are the different properties that state the type of styling you want a particular element to have and what the value is. If this would be stated in plain English, it would read something like: I want to select the h1 tags. (selector) and I want their color to be green. --> h1{ color: green; }.
    Notice how the declaration is siting between two curly braces. All the declarations of a selector are included between two curly braces and they are separated by semi-colons (;).
  • Property: The property is the entity that states what needs to change (eg: color, positioning, background, display). It is the first part of the declaration, before the colon symbol (:) --> h1{ color: <-- The property is the one to the left; }
  • Value: The value is the entity that states how the property changes / what the value of the property is. --> h1 { The value is the one to the right --> green; }

In the following example, you can see how the declarations are being entered in a new CSS rule. In this example, all the <p> nodes have a font color of green and the font size is 16 pixels.

p { color: green; font-size: 16px; }
